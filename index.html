PDV - Ponto de Atendimento

⚡ Principal

O PDV é o módulo principal para realizar vendas. Ele foi projetado para ser rápido, intuitivo e funciona mesmo sem internet.

🧾 CUPOM 3 itens
2x Coca-ColaR$ 8,00
1x HambúrguerR$ 25,00
3x Batata FritaR$ 15,00
TOTALR$ 48,00
💵 F1 ⚡ F2 💳 F3 💳 F4 📄 F5 ✅ F9

🎯 Como Usar

1. Busque o produto pelo nome ou código de barras

2. Clique no produto para adicionar ao carrinho

3. Ajuste quantidades com + e -

4. Escolha a forma de pagamento (F1-F5)

5. Finalize com F9

6. Imprima o cupom (PDF ou Bluetooth)

Perguntas Frequentes (FAQ)

❓ Dúvidas
Q Posso usar o sistema sem internet?
Sim! O PDV funciona offline. As vendas são salvas localmente e sincronizadas quando a internet voltar.
Q Como faço backup automático?
O sistema gera relatórios diários automaticamente via Celery. Para backups completos, use o módulo "Backup".
Q Como configurar comissões?
Vá em "Comissões" > "Configurar" > Selecione o vendedor > Defina o percentual.
Q O sistema é seguro?
Sim! O sistema usa JWT, bcrypt, rate limiting, CORS configurado e todas as boas práticas de segurança.
Q Como funciona o split payment?
No PDV, clique em "Split" e adicione valores para diferentes formas de pagamento (ex: R$ 50 dinheiro + R$ 50 cartão).
Q Posso ter múltiplas filiais?
Sim! O sistema suporta múltiplas filiais com dados separados.
